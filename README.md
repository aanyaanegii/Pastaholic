# Pastaholic 🍝

A beautifully designed pasta restaurant website created using HTML and CSS.

## 📌 About the Project

Pastaholic is a frontend restaurant website that showcases a modern and elegant design for a pasta-specialty restaurant. The website includes a landing page, menu section, about section, and contact information with a visually appealing layout.

## ✨ Features
- Attractive hero section
- Pasta menu showcase
- About restaurant section
- Contact section
- Smooth navigation between sections
- Modern UI design

## 🛠️ Technologies Used

- HTML5
- CSS3



  
## 🚀 How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Explore the Pastaholic website.
